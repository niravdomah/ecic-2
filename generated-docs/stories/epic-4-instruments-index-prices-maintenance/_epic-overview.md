# Epic 4: Instruments & Index Prices Maintenance

## Description
Implements CRUD screens for managing instruments master data and index prices. Users can view, create, edit, and delete instruments with full audit trails showing who made changes and when. Outstanding items requiring attention are highlighted. This epic provides the foundation for maintaining reference data used in risk calculations.

## Stories
1. **Instruments List View** - Display searchable/filterable table of all instruments with CRUD actions | File: `story-1-instruments-list.md` | Status: Pending
2. **Instrument Create/Edit** - Add new instruments or edit existing ones via modal form with validation | File: `story-2-instrument-create-edit.md` | Status: Pending
3. **Instrument Delete** - Remove instruments with confirmation dialog and constraint checking | File: `story-3-instrument-delete.md` | Status: Pending
4. **Index Prices Maintenance** - Manage index prices with similar CRUD functionality and audit trail | File: `story-4-index-prices-maintenance.md` | Status: Pending
