# Feature: InvestInsight Platform

## Summary
InvestInsight is an investment risk analysis platform that manages the monthly workflow for importing portfolio data, validating completeness, maintaining reference data, and producing approved reports. The system handles multi-source data imports (Portfolio Manager, Custodian, Bloomberg), data quality validation, multi-level approvals, and comprehensive process logging.
