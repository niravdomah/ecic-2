# Epic 1: Start Page & Batch Management

## Description
Establishes the InvestInsight home page as the central dashboard for batch management. Users can create new monthly batches, view current batch status, and navigate to all major feature areas. This epic replaces the template placeholder and implements the foundation for the entire application workflow.

## Stories
1. **Home Page Setup** - Replace template placeholder with InvestInsight start page | File: `story-1-home-page-setup.md` | Status: Pending
2. **Batch Creation Modal** - Create new monthly batch with batch number and month selection | File: `story-2-batch-creation-modal.md` | Status: Pending
3. **Current Status Display** - Show batch progress across workflow stages with visual indicators | File: `story-3-current-status-display.md` | Status: Pending
4. **Quick Navigation** - Provide shortcuts to File Uploads, Data Confirmation, Maintenance, and Admin areas | File: `story-4-quick-navigation.md` | Status: Pending
