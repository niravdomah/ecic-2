# Epic 5: Risk Data Maintenance

## Description
Implements CRUD screens for managing risk-related reference data: durations, betas, and credit ratings. Each screen includes outstanding items tracking to highlight instruments requiring data updates. Users can filter by outstanding status, perform bulk updates, and maintain full audit trails for compliance.

## Stories
1. **Durations Maintenance** - Manage duration values for instruments with outstanding items tracking | File: `story-1-durations-maintenance.md` | Status: Pending
2. **Betas Maintenance** - Manage beta coefficients for risk calculations with bulk update capability | File: `story-2-betas-maintenance.md` | Status: Pending
3. **Credit Ratings Maintenance** - Manage credit ratings with agency selection and history tracking | File: `story-3-credit-ratings-maintenance.md` | Status: Pending
4. **Outstanding Items Filter** - Filter and highlight instruments missing required risk data | File: `story-4-outstanding-items-filter.md` | Status: Pending
